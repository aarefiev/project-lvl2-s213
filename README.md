Compares two configuration files and shows a difference.

## Setup

```
$ make install
```

## Usage

```bash
$ gendiff [options] <firstConfig> <secondConfig>

Options:

  -h, --help           output usage information
  -V, --version        output the version number
  -f, --format [type]  Output format
```
